
# Elementary Tweaks

Caso queira utilizar o Elementary OS com algumas opções a mais de customização, siga os passos abaixo: 

## Instalação

Você tem que primeiramente adicionar o suporte a PPA, para isso, vamos utilizar o terminal e dar o seguinte comando:
```
$ sudo apt install software-properties-common_
```

Depois, devemos instalar o Elementary Tweaks, então teremos que adicionar o repositório PPA e depois baixar o Elementary Tweaks:
```
$ sudo add-apt-repository -y ppa:philip.scott/elementary-tweaks_
```
```
$ sudo apt update
```
```
$ sudo apt install -y elementary-tweaks
```
E pronto, agora só aproveitar :)

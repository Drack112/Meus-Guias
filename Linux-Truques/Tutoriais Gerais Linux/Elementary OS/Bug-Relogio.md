# Bug Do Relogio

Provalvemente isso vai acontecer com você mais cedo ou mais tarde no Elementary OS. Ha um pequeno bug onde simplesmente o relógio do elementary some e não tem como ajustar de forma grafica.

Então aqui vai alguns comandos que você tera que realizar para resolver isso.


## Comandos

```
$ sudo apt install --reinstall wingpanel-indicator-datetime
```

## O que é esse comando?

Simples! Ele apenas ira reinstalar seu datetime e então seu relógio voltara a ser exibido no painel ;-)
